# Phantom Docs

This directory is organized around the current supported Phantom surface on Zig `0.17.0-dev`.

## Start Here

1. `../README.md`
2. `getting-started/integration.md`
3. `reference/api.md`

## Layout

- `getting-started/`
  - `integration.md`: recommended way to adopt Phantom
- `reference/`
  - `api.md`: current public API summary
  - `features.md`: feature/status overview
  - `widget-inventory.md`: widget catalogue
  - `roadmap.md`: planning notes
- `guides/`
  - `async-streaming.md`
  - `grove-integration.md`
  - `search.md`
  - `terminal-sessions.md`
  - `themes.md`
  - `unicode.md`
  - `workspace-apps.md`
- `architecture/`
  - `rendering.md`
  - `transitions.md`
- `widgets/`
  - `widget-guide.md`
  - `migration-v061.md`
- `rfcs/`
  - RFCs and longer-term design notes

## Naming Policy

- Keep `docs/README.md` uppercase.
- Use lowercase kebab-case for all other markdown files in `docs/`.
